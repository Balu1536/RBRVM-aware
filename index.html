About the Project

Reverse Vending Machine Initiative

Our RVM project aims to automate the segregation and recycling of plastic and metal waste. Users can deposit their recyclable items into the machine, which sorts the waste and rewards them with digital currency called Supercoins. These coins can be redeemed online via UPI-linked accounts.

The system uses advanced sensors to detect the type of material, calculate weight, and distribute appropriate rewards. The website complements the machine by offering educational content, quizzes, and user dashboards.
